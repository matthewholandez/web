# Personal Website

A personal portfolio website built with [Next.js 16](https://nextjs.org/).

## Features

- **Next.js 16**: App Router and Server Components.
- **TypeScript**: Fully typed codebase.
- **Tailwind CSS**: Utility-first styling.

## Prerequisites

- Node.js 18+ installed.

## Setup Instructions

### 1. Clone the repository

```bash
git clone https://github.com/matthewholandez/web.git
cd web
```

### 2. Install dependencies

```bash
npm install
```

## Running the Project

Start the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Project Structure

- `app/`: Next.js App Router pages and layouts.
  - `page.tsx`: Home page.
- `components/`: Reusable UI components.
- `lib/`: Data and utilities.
- `public/`: Static assets.

## Deployment

This project can be easily deployed on [Vercel](https://vercel.com/).

1. Push your code to a Git repository.
2. Import the project into Vercel.
4. Deploy.
