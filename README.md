## Structure

- **Homepage** located in `apps/home`, a Next.js application hosted on Vercel
- **Blog** located in `apps/blog`, built with Jekyll and hosted on GitHub Pages

## Start

```bash
npm run dev

# Or, to start only the homepage
npm run dev:home

# Or, to start only the blog
npm run dev:blog
```
